Una de las estructuras de datos que podemos utilizar para conseguir esta finalidad, son las colecciones. Existen dos formas de declarar una colección, una de ellas es declarar una instancia Zion del objeto a raíz Y otra de ellas consiste en inicializar una nueva colección simplemente poniendo dos corchetes. Al realizar esa operación, automáticamente el intérprete entiende qué es identificador se convierte en una colección. A continuación, podemos introducir un nuevo elemento dentro de esa colección de datos, de la siguiente forma. Llamaremos al identificador cómo se llamar a MOLLÀ una variable, pero introduciremos los corchetes, y de todos los corchetes, introduciremos un valor numérico. Aunque podemos empezar en cualquier valor numérico, se recomienda que nos lo primero índice empiece el número cero. A continuación, introducimos un segundo elemento dentro de la colección, en este caso, seleccionando un número diferente para el identificador, es decir, para el índice. Sin el ejemplo anterior hemos utilizado el número cero, en este caso utilizaremos el número uno, y le asignaremos un valor diferente a la posición cero de la colección.
