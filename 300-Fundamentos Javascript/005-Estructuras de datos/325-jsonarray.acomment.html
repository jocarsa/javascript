no solo podemos crear objetos dentro de JavaScript utilizando la sintaxis J son, sino que también podemos combinar esta estructura de datos con la estructura anterior, que es la colección. Así pues, en este ejemplo, se muestra un código mediante el cual introducimos, dentro de un objeto J, son, una colección de datos, y dentro de cada elemento de la colección, introducimos, a su vez, otro J son. Esta es la segunda de las ventajas de las que hacía referencia anteriormente dentro de este mismo capítulo, con respecto a aquellas ventajas que nos proporciona el uso de objetos J son, con respecto a las colecciones. puedo tener un nivel limitado de complejidad en mis colecciones, de tal forma que cualquier estructura de datos es posible utilizando el formato J son
