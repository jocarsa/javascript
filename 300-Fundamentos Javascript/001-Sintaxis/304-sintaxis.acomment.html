Si bien el lenguaje HTML es bastante permisivo en el sentido de que podemos escribir prácticamente lo que queramos, y donde queramos, el JavaScript no ocurre tanto así, y este lenguaje de programación tiene una serie de reglas, bastante más estrictas de sintaxis, que debemos respetar, ya que, en caso contrario, el script que estemos desarrollando, nos dará un error. En este primer ejemplo de código, vamos a realizar un código erróneo, ya que, dentro del cuerpo del documento HTML, escribiremos una primera línea que nos anunciará que el texto lo habremos escrito en el lenguaje HTML. después de la línea de apertura de la etiqueta descrita, escribiremos otro texto que nos dirá que ese texto vuelve a ser HTML. Pero, sin embargo, dentro de la etiqueta, script, escribiremos texto libre, cuando realmente no podemos realizar esa acción una vez que hemos empezado a escribir código JavaScript. Por lo tanto, si intentamos ejecutar ese código dentro del navegador, comprobaremos cómo proporciona un error, ya que esa línea no es un código válido en el lenguaje Java script.
