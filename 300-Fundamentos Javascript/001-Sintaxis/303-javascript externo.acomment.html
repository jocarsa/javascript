A continuación, dentro del archivo HTML, con el que estaremos trabajando dentro de este ejercicio, vamos a realizar una llamada, utilizando la misma etiqueta, escrita que hemos utilizado en el ejemplo anterior, pero en este caso, en lugar de que la propia etiqueta script tenga código dentro de la misma, vamos a realizar una llamada a un archivo externo, concretamente, al mismo archivo que hemos creado anteriormente, utilizando el parámetro SRC de la etiqueta, HTML. -- De esta forma, si ejecutamos el código, vamos a poder comprobar como se ejecutará exactamente igual que en el ejemplo anterior, la diferencia es que, en este caso, en lugar de tener todo el código reunido, dentro de un mismo archivo, lo tenemos separado en dos archivos diferentes. Esta forma de programar y esta forma de escribir código, en un entorno web, para ejemplos sencillos, puede resultar confusa, ya que tendremos que estar trabajando en todo momento con dos archivos diferentes. Sin embargo, de la misma forma que es una regla general en el desarrollo informático, cuando un proyecto crece, el proyecto se volverá mucho más manejable si separamos el código en archivos diferentes, especialmente cuando la función del código es diferente, en el sentido de que, para este caso, HTML consiste en la estructura del documento, mientras que el JavaScript consiste o corresponde a la interacción y a la programación que va haber asociada dentro de esa página web
