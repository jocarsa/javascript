una de las formas más sencillas y más comunes de introducir código JavaScript, dentro de una página web, consiste en introducirlo de forma interna. Al introducirlo de forma interna, creamos una etiqueta escrito dentro del texto, y, dentro de esa etiqueta, escribimos todo el código que tenga nuestro programa desarrollado en JavaScript. esta es la forma que vamos a utilizar para trabajar en todo este primer apartado de fundamentos del lenguaje JavaScript, ya que es cómoda desde el punto de vista en el que nos permite tener todo el código introducido dentro de un mismo archivo, y no tener que trabajar con diferentes archivos de forma simultánea. Sin embargo, para proyectos grandes, es mucho mejor separar el código en archivos diferentes, puesto que dificulta la estabilidad escalabilidad, tener código de diferentes lenguajes dentro de un mismo archivo.
